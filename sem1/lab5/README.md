# Физический маятник 

### Цель работы 

Изучить гармонические колебания на примере движения
физического маятника. Определить момент инерции физического маятника
методом колебаний.

### Важные замечания из теории

У нас есть барабан и стержень массами $m_1,m_2$ соотвественно и грузы одинаковой массы $m_3$, которые мы можем перемещать по стержню.

Заметим, что если перемещать груз относительно некоторой точки на одинаковое расстояние, то центр масс не изменится.

Однако момент инерции $I$ маятника будет меняться по закону 

$$ I=I_A + 2m_3 r^{2} $$

Давайте заметим, что после разных преобразований мы получим некоторую функцию $T^2=f(r^2)$

$$T^2= \frac{4 \pi}{g}\left(\frac{I_A}{Ml} + \frac{2r^2m_3}{Ml} \right)$$

- $M=m_1+m_2+m_3$

Теперь мы получим некоторую кривую, которая описывает $y=kx+b$, после чего мы сможем найти $I_A$
